# Changelog

## [Unreleased](https://github.com/byheaven/byheaven-skills/compare/xhs-publisher-0.2.0...HEAD)

## [0.2.0](https://github.com/byheaven/byheaven-skills/compare/3a99986b5b873299416f618ef4f30044e030ce4e...xhs-publisher-0.2.0) (2026-03-18)

**OpenAI Codex support** — the skill now includes Codex app metadata (`agents/openai.yaml`
and favicon), making it discoverable and usable in OpenAI Codex and other compatible AI tools
in addition to Claude Code.

## [0.1.0](https://github.com/byheaven/byheaven-skills/commit/3a99986b5b873299416f618ef4f30044e030ce4e) (2026-01-23)

**Automated Xiaohongshu publishing** — publish Xiaohongshu posts directly from
your AI workflow using either note files or inline content, without manually
copying text into the creator dashboard.

**Two publishing paths** — the initial release supports both shorter image-post
flows and longer article-style note flows, so the plugin can choose the right
editor experience for the content you are preparing.
